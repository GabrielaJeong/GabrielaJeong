## 🧪 currently playing with

### Folio

**Folio** is my AI character chat project.

It started with a simple question:

> can I make a real product-ish thing with AI coding tools?

So I’m building it piece by piece:

- character personas
- tiny worlds and relationship settings
- prompt structure
- conversation memory
- guardrails
- admin tools
- frontend and design system
- Cursor / Claude workflow

It’s part product lab, part character playground, part “let’s see if this actually works.”

## 🧶 little beliefs

- messy ideas are not bad; they just need structure
- tiny prototypes teach faster than perfect documents
- characters are interfaces too
- AI tools are better when you give them good rails
- PMs don’t need to code everything, but it helps to understand how things break
